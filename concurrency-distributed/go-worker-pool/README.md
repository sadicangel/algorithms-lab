# Worker Pool with Goroutines

Implement a worker pool pattern using lightweight threads or goroutines.

## Tasks

- Create a job channel/queue and a set of worker routines.
- Support graceful shutdown and cancellation via context or signals.
- Measure throughput for different pool sizes and workload characteristics.

## Suggested Languages

- Go
- C#
- Kotlin

## Real-World Use Cases

- Parallelizing CPU-bound tasks or I/O-bound operations.
- Simple background job processing service for web applications.
